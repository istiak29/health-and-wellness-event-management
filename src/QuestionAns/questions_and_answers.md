###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A

<i>Because we declared a variable but did not gave any value to it. the console.log will give us an empty object. since we create an empty object</i>

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A

<i>Because the variable info contain an object which the first value of food. Here the object key favoriteFood only contain the the first value of food array not the reference of the array. So when we change the value of favoriteFood we are only changing the value it will not affect the array.</i>

</p>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {

}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B

<i>sayHi function takes one parameter but we did not provide it, so by calling the function it will receive null. For null value return will be undefined.</i>

</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript

});

```

- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>forEach function checking whether the value of nums array is truthy or not. since 0 is not a truthy value in javaScript so the answer is 3. Because every time if the nums value is truthy then count value will increase.</i>

</p>
